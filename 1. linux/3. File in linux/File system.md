## File system

<!-- Core system folder -->

- `bin`: Commands by user (ls, cp, mv, rm)
- `sbin`: Commands by root (mount, fsck, reboot)
- `boot`: Boot loader files
- `dev`:
- `etc`: Configuration
- `lib`: Essential shared libraries
- `lib64`
- `proc`: Process information (Virtual Filesystem)
- `run`
- `sys`
- `tmp`: Temporary files
- `usr`: User programs and data
- `var`

<!-- User folders -->

- root (root user)
- home/<username>

<!-- Mount Points and Optional Software -->

- media
- mnt
- opt
- srv
